# 🧠 Memory Fun!

A fun and educational memory game for kids aged 5–10. Six game modes, three difficulty levels, sounds, spoken words, and a confetti celebration when you win!

## Game modes

| Mode | What kids practice |
|---|---|
| 🎵 Musical | Listening — cards look the same, match them by their tune |
| 🔷 Shapes | Shape and color recognition |
| 🐶 Pictures | Visual memory with animals |
| 📖 Words | Reading easy words (the game reads them aloud!) |
| 🔢 Numbers | Counting — match the number to the right amount of stars |
| 🧩 Word + Picture | Early reading — match the word "cat" to 🐱 |

Difficulty: Easy (6 cards), Medium (12 cards), Hard (20 cards). Best scores are saved on the device.

## How to publish it with GitHub Desktop

1. Open **GitHub Desktop** → **File → New Repository**. Name it `memory-fun`, and set the **Local Path** to a folder of your choice, then click **Create Repository**.
2. Copy all the files from this folder (`index.html`, `manifest.json`, `sw.js`, the 3 `.png` icons, `README.md`) into that new repository folder.
3. Back in GitHub Desktop you'll see the files listed as changes. Type a summary like "First version" and click **Commit to main**.
4. Click **Publish repository**. Uncheck "Keep this code private" (GitHub Pages on a free account needs a public repo), then click **Publish Repository**.
5. Go to the repository on **github.com** → **Settings** → **Pages** (left sidebar). Under **Branch**, choose `main` and `/ (root)`, then click **Save**.
6. Wait a minute or two. Your game will be live at:
   `https://YOUR-USERNAME.github.io/memory-fun/`

## How to put the icon on a phone or iPad

**iPad / iPhone (Safari):**
1. Open the game link in Safari.
2. Tap the **Share** button (square with an arrow).
3. Tap **Add to Home Screen** → **Add**.
4. The Memory Fun icon appears on the home screen and opens full-screen like a real app — it even works offline after the first visit.

**Android (Chrome):**
1. Open the game link in Chrome.
2. Tap the **⋮** menu → **Add to Home screen** (or "Install app").

## Tips

- The 🎵 Musical mode needs sound on — check the mute switch on the iPad!
- Words are spoken aloud automatically to help early readers.
- To update the game later: change the files, commit, and push in GitHub Desktop — the website updates itself.
